## Getting Started

To get a local copy up and running clone this repository to your
computer.

### Prerequisites

Gatsby: >=2.2

### Usage

Start server with:

```
  gatsby develop
```

Open `http://localhost:8000/` in your browser.

### Build static website

```
  gatsby build
```
